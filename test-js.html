Prueba de JavaScript

Si el HTMl proporciona el estilo del documento, el CSS la estructura visual, entonces es JavaScript el que proporciona la interactividad de la web con el ususario.

con JS podemos hacer que la página reacciona ante distintas acciones del usuario. Hacer clic, pasar el ratón sobre un objeto en concreto, hacer scroll, usar teclado etc

al igual que HTML nos provee la etiqueta style para insertar el CSS. Tenemos la etiqeuta Script para escribir JavaScript. El código también puede incorporarse como atributo de los elementos (como sucede con el atributo style) por lo que su uso dependerá del criterio del desarrollador.
