id's y clases

las id y las clases son elementos que se pueden asignar a los elementos de HTML. Mientras que as clases se pueden asignar a varios elementos, las id son únicas ysolo pueden tenerla un elemento.

Id's

Los id's apuntan a un único elemento. No puede haber más de una elemento con la misma i'd.

Sintaxis: En CSS se declara con #. No puede empezar por número y no puede contener espacios en blanco
