Objetos en JavaScript

Un objeto es un tipo especial de variable, que almacena distintos valores clasificados como propiedades

Un objeto también puedes almacenar distintos metodos. Estos son bloques de código que se ejecutarán en determinados momentos.

En la vida real, un coche es un objeto.

Un coche tiene propiedades como peso y color, métodos como arrancar y parar.

Todos los coches tienene las mismas propiedades, pero estás tienen un valor distinto dependiendo del coche.

Todos los coches tinen los mismos métodos, pero estos sepueen ejecutar en distintos momentos.

Normalmente, usamos una variable para guardar un valor sencillo:

let coche: 'Renault'

Los objetos son variables que pueden contener multipls valores: 

        const coche = {
            marca: "Renault",
            modelo: "magane",
            color: "gris" 
        }

Los valores se escriben emparejados con el nombre de la propiedad name: value.
