"Layout" en CSS

El layout es la disposición o la forma en que los elementos se muestran en pantalla según su tipo.

Display

Es la propiedad más importante del CSS para controlar el alineamiento.

La propiedad Display especifica la forma en que se muestra un elemento.

En HTML cada elemento posee un valor por defecto dependiendo del tipo de elemento que sea. La mayoría de los elementos una propiedad block o inline por defecto.

Diferencias entre display: none y visibility: hidden

Al ocultar un elemento con display: none la página se mostrará como si el elemento no estuviera ahí.

Al ocultar un elemento con visibility: hidden el elemento no se mostrará pero el espacio que ocupa quedará en blanco.

La propiedad position

Esta propiedad define el metodo de posicionamiento de un elemento.

Hay 5 tipos de posición.

  • static
  • relative
  • fixed
  • Absolute
  • sticky

Los elementos se posicionan con las propiedades top, left, right, bottom. Sin embargo estas propiedades no funcionarán sin especificar primero la position. Además, dependiendo del tipo position las propiedades funcionarán de manera distinta.

position: static

Todos los elementos tienen este valor por defecto

Las propiedades top, bottom, left y right no afectan a los elementos con posición estática.

position: relative

Un elemento con este valor se posicionará de manera relativa a su posición normal. Es decir, el centro del eje será sobre si mismo.

este div tiene position: relative

position: fixed

Un elemento con este valor se posicionará relativamente respecto a la pantalla. Lo que quiere decir que tendrá la misma posición incluso si scrolleamos la web.

position: absolute

Un elemento con este valor se posicionará al respecto a su contenedor posicionado (no estático).

Si un elemento absoluto no tiene ancestros posicionados, usará el body y actuará como un position: fixed

position: sticky

Un elemento sticky alternará su comportamiento entre relative y fixed dependiendo de la posición del scroll de la página.
