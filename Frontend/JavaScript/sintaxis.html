variables

En programación las variables se usan para almacenar valores.

JavaScript se sirve de las keywords var, let, const para declarar variables.

el signo = se usa para asignar valores a las variables.

Operadores

JavaScriptusa operadores aritméticos (+ - * /) para computar valores

Expresiones

Las expresiones son una combinación de valores, variables y operadoresque computan en un resultado o valor. Ref al js linea 11

Comentarios

Los comentarios son líneas de código que son ignoradas no se ejecutan.

Tienen principalmente dos usos. El primero es explicar el código y el segundo es para desactivar cierta fase de nuestro código sin necesidad de borrarla.

En JavaScript hay dos maneras de comentar. En línea con // y en bloque con /*...*/

Identificadores/Nombres

Los Identificadores se usna para nombrar variables y funciones

Hay una serie de reglas para nombrar comunes en la mayoría d elenguajes de programación.

Un identificador debe empezar por:

  • una letra(A-Z o a-z)
  • Un simbolo de dolar
  • o una barra baja

Los caractreses siguientes pueden sear números letras, _, $.

No se permite empezar con números

Uso de mayúsculas

Los Identificadores son sensibles a las mayúsculas. Esto quiere decir que lastName y lastname son dos variables diferentes

Y, por ejemplo, JavaScriptno interpreta LET o let como la palabra clave let

Camel Case

Antes los programadores usaban guiones (-) para unir distintas palabras en un identificador. last-name, first-name...

Sin embargos, en JavaScript los guiones están reservados para las restas. Por los programaors de JS se inclinan a separar las distintas palabras mediantes mayúsculas, dejando todo en minúsculas.

primerApellido, colorVehiculo, codigoPostal
