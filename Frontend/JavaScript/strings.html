Strings con JS

Un string es cero o más caracteres escritos dentro de comillas

Esto lo convierte básicamente en una cadena de caracteres

Se pueden usar tanto comillas simples como dobles. Un string puede contener comillas siempre y cuando sean distintas de las que rodean el string:

let str = 'Tere "Terere' Herrera'

Si imprimos este string: Tere "Terere' Herrera.

Métodoos de strings

Lenght

Para obtener la longitud de un string se usa el método lenght()

let txt = 'ABCDEFGHIJKLMNOPQRSTUVWXYZ'
let longitud = txt.lenght();

slice()

Corta o extrae una part de un string y la retorna como un nuevo string.

slice() necesita dos párametros. La posición inicial y la final (no incluída en el corte).

substr()

Funciona similar al slice(), pero el segundo parametro índica la longitud del string a recortar. Deprecado

replace()

Reemplaza una parte de un string y lo retorna como un string nuevo.

El primer párametro es el valor a buscar en el string y el segundo es el valor por el que lo reemplazamos.

estos valores son case sensitive. Es decir, que los valores con mayúsculas y minúsculas deben coincidir.

solo se reemplaza la primera coincidencia en caso de haber varias en el string.

Para sortear la diferencia de las mayúsculas, se puede usar una expresión regular.

Transformar a minúsculas y viceversa

Podemos usar toLowerCase() o toUpperCase() para transformar un string o una parte en minúsculas o mayúsculas, respectivamente.

Limpiar strings

Podemos usar trim() para limpiar los espacios en blanco redundantes de un string.

Si tenemos un str nombre = "Román" el trim lo dejará como "Román".

CharAt()

Retorna el cáracter ubicado en el índice especificado como párametro.

Convertir string en arrays con split()

Si tenemos un string cuyos valores están separados por comas o cualquier otro caracter distinguible, podemos servirnos de esto para transformarlo en un array.
