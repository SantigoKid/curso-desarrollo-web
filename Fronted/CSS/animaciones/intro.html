Animaciones con CSS

CSS permite animar elementos HTML sin necesidad de recurrir JavaScript

Qué son las animaciones?

Una animación es el cambio gradual de un estilo a otro.

Se puede cambiar tantas propiedades de CSS como se quiera, tantas veces como sea necesario.

Para usar una animación debe especificarse la regla @keyframes

La regla @keyframes

Esta regla contiene los estilos que el elemento tendrá en determinados momentos.

Cuando se especifica una regla, los cambios se ejecutarán de manera gradual en ciertos momentos.

Para que la animación funcione, debe asignar un keyframe al CSS de un elemento.

Propiedades de las animaciones

  • animation-delay
  • animation-iteration-count
  • animation-direction
  • animation-timing-function
  • animation-fill-mode

animation-delay:

Especifica el retardo de una animación.

Los valores positivos indican el tiempo que ha de transcurrir para la animación de comienzo. Los valores negativos indican que la animación empezará como si ya hubiese transcurrido el tiempo especificado.

animation-iteration-count

Establece el número de veces que se repetirá una animación

Se puede establecer un número natural de veces o se puede indicar un número ilimitado con el valor infinte.

animation-direction

Especifica si la animación se producirá en orden normal, inverso o irá alternando ciclos. Los valores posibles son:

  • normal La animación se produce hacia delante. Valor por defecto.
  • reverse La animación hacia atras (de 100% a 0%).
  • alternate La animación se produce hacia delante, y luego hacia atrás.
  • alternate-reverse La animación se produce hacia atrás, y luego hacia delante.

animation-timing-function

Escpicifica la curva de velocidad al reproducir una animación.

Tener los siguientes valores:

  • ease La animación tendrá un inicio lento, luego irá rápido y tendrá un valor por defecto
  • linear La animación tendrá una velocidad estática durante toda a duración
  • ease-in La animación tendrá un inicio lento.
  • ease-out La animación tendrá un final lento.
  • ease-in-out La animación tendrá un inicio lento y un final lento
  • cubic-bezier(n,n,n,n) Permite al usuario especificar la curva de velocidad mediante una función.
