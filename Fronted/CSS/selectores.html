Selectores de CSS

Los selectores tienen la función de encontrar o seleccionar elementos HTML a los que queremos dar estilo

  • Selectores sencillos - Apuntan a elementos basandose en su clase ., id # o nombre h1
  • Combinadores - Selectores basados en una espécifica relación entre los elementos .contenedor div
  • Pseudo-clases - Son selectores basados del estado del elemento button:hover
  • pseudo-elementos - Selecciona una parte especifica de una elemento
  • De atributo - Seleccione elementos en un atributo o en el valor de un atributo

Selectores de nombre de elemento

Selecciona los elementos basandose en su nombre de elemento.

Ejemplo: los span serán de color morado . 

                span {
                    color: purple;
                }

Selectores de ID

Este selector usa el atributo id de un elemento HTML para seleccionar Un elemento especifico

la id de un elemento es única en un documento. Por lo tanto, el selector de id # se usa para especificar un único elemento.

Ejemplo: Este párrafo será el único en estar centrado y de color verde limon

El selector universal

El selector universal * selecciona a los elementos.

Selector de agrupación

Sirve para agrupar aquellos selectores que tienen las mismas características

Ejemplo: Digamos que queremos darle la misma sombra que .pSombras a todos los h1 y h2. En vez de ir a todos elementos h1 y h2 y darles esta clase podemos hacer lo siguiente; 

        
            .pSombras {
                box-shadow: 5px 5px 3px -2px;
            }
            h1 {
                box-shadow: 5px 5px 3px -2px;
            }
            h2 {
                box-shadow: 5px 5px 3px -2px;
            }

Agrupado quedaría así:

        
            .pSombras, h1, h2 {
                box-shadow: 5px 5px 3px -2px;
            }
