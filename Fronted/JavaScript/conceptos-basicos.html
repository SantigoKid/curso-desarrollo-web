Conceptos basicos de JS

Statements o declaraciones

Un programa informatico es una lista de instrucciones que se ejecutan por un ordenador

En programación estas instrucciones se llaman declaraciones o Statements

un programa de JavaScript es una lista de declaraciones de programación

Statements o declaraciones

las declaraciones se componen de valores (value), operadores(operator), palabras claves(keywords), y comentarios (comments).

Ejemplo: Tenemos una declaración(Statement) que le indica al navegador que debe Escribir "Hola Mundo" dentro de un elemento de HTML con la id="test"

document.getElementById('test').innerHTML = 'Hola Mundo'

los Statements, o línes de código se ejecutan una a una en un orden de lectura.

punto y coma (semicolon)

se usan para separar los statements en JS. No son obligatorios pero si muy recomendados, se usan por buenas practicas de código. Ref al JS línea 1.

Espacios en blanco

JavaScript ingora los espacios en blanco y se usan para hacer el código más legible.

let persona = "Carlos";
let person="Carlos";

Bloques de código

Las declaraciones o statements se pueden agrupar en bloques dentro de llaves (curly brackets) {...}

El proposito de ésto, es definir una serie de declaraciones para que se ejecuten juntas.

Un ejemplo de declaracionesunidas en bloques son las funciones. Ref al js 16

Plabras clave o keywords

A menudo, muchas líneas de código empiezan por una palabra que indican una acción a realizarse

Estas palabaras están reservadas y no pueden usarse en ningún otro contexto

  • var- Declara una variable
  • let- Declara una variable de bloque
  • const- Declara una constante de bloque
  • if- Indica un bloque de código a ejecutarse cuando se cumple una condición
  • Switch- Indica un bloque de código a ejecutarse dependiendo de distintos casos
  • for- Indica un bloque de código a ejecutarse en bucle
  • function- Declara una función
  • return- Sale de una función
  • try- Implementa manejo de errores a un bloque de código
