Todo sobre let

La palabra clave let o keyword let tiene 3 usos principales

  • Las variables declaradas con let no puedens er declaradas.
  • las variables definidas cont let deben declararse antes de ser usadas.
  • Las variables delcaradas con let tienen un alcance (scope) de bloque.

No se puede redeclarar

En JavaScript al usar var se puede, accidentalmente, redeclarar y cambiar el valor de una variable.

var x = 'Juan';
var x = 0;

let impide redeclarar variables y por tanto, es más seguro

Alcance de bloque (block scope)

Las variables declaradas con let tienen alcance de bloque. Es decir, que aquellas variables declaradas dentro de un bloque {} no pueden ser usadas fuera del bloque . Red al Js línea 1
