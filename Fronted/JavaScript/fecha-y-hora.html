Fecha y hora en JavaScript

El objeto fecha date()

Nos permiten trabajar con las fechas

El objeto Date() es estático. No es un reloj como tal , es un objeto cuyas propiedades se generan cuando el navegador carga el documento.

Creando Objetos fecha

Los objetos de fecha se crean con new Date()

Hay diferentes maneras de crear uno. Si no le indicamos parámetro, el objeto se genera con la fecha y hora actuales.

  • new Date()
  • new Date("string de fecha")
  • new Date(año, mes, día, hora, minutos, segúndos y ms)
  • new Date(milisegúndos)

new Date('string de fecha')

Crear un objeto a partir de una fecha concreta. En este caso será el 7 de febrero del 99. Ref a JS en línea 4.

new Date(año, mes...)

La fecha se crea introduciendo números. Esto se genera con ciertas situaciones especiales. Ref en linea 11.

Nota JavaScript ceunta los meses de 0 - 11. Si se introduce un mes mayor que 11, JS seguirá contando los meses del siguiente año.

New date (milisegundos)

JavaScript almacena las fechas como el número de ms que transcurrido desde el 1 de enero de 1970.

El tiempo 0 (Zero Time) es el 1 de enero de 1970, 00:00:00 GTM 0.

Ahora el tiempo/fecha es: milisegundos transcurridos en tiempo cero.

Tomar datos de fecha (Get Methods)

  • getFullYear() - Retorna el año como un número de 4 dígitos YYYY
  • getMonth() - Retorna el mes como número 0- 11
  • getDate() - Retorna el día del mes como número 1 - 31
  • getDay() - Retorna el dia de la semana como número - 6
  • getHours() - 0 - 23
  • getSeconds() - 0 - 59
  • getMiliseconds() 0 - 999
  • getTime() Retorna el tiempo o los ms desde el tiempo cero.
